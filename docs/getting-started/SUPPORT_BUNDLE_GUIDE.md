# Support Bundle Guide

## When to export

- unexpected ingest drop
- repeated action failures
- contradictory console vs CLI observations

## What to include

- support bundle archive
- `mel doctor` output
- relevant timeline window
- incident IDs and action IDs
- sanitized config snippet
