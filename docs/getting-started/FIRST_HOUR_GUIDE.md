# First Hour Guide

## 0–15 min

- Build and run MEL locally.
- Confirm doctor output and transport posture.

## 15–30 min

- Review incidents and timeline surfaces.
- Validate support bundle export path.

## 30–45 min

- Exercise control workflow in advisory mode.
- Inspect pending approvals and audit states.

## 45–60 min

- Run a mock incident drill.
- Export proofpack/support bundle.
- Document unresolved uncertainties.

## Exit criteria

You can explain:

- what MEL knows from direct evidence
- what MEL infers (advisory)
- what MEL explicitly does not claim
