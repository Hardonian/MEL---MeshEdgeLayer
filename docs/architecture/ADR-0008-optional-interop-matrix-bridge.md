# ADR-0008-optional-interop-matrix-bridge: Interop is optional via bridge provider

## Status
Accepted

## Decision
Federated chat/interop remains optional through provider interfaces (such as Matrix bridge) and cannot become mandatory for core MEL operation.

## Consequences
- Reduces lock-in and recurring spend for core MEL operation.
- Keeps operator-truth claims bounded to locally verifiable evidence.
- Preserves swap-friendly provider seams for future implementation changes.
