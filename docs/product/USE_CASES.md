# Use Cases

## 1) First mesh incident triage (single operator)

- Identify whether data is live or stale.
- Pull transport and node evidence.
- Build a timeline and export proofpack for handoff.

## 2) Control-path governance under pressure

- Keep auto-remediation bounded by policy.
- Require approvals for high blast-radius actions.
- Preserve actor attribution and decision notes.

## 3) Support escalation with minimal ambiguity

- Generate support bundle.
- Include diagnostics + config validation + timeline snippets.
- Distinguish known limitations from runtime failures.

## 4) Post-incident learning loop

- Capture incident class and remediation decision.
- Record observed outcome and residual uncertainty.
- Feed runbooks and readiness checklists for recurrence reduction.
