# Recommendation Intelligence Roadmap

This roadmap separates shipped capabilities from aspirations.

## Shipped (today)

- Advisory recommendations bounded by deterministic evidence and policy checks.
- Explicit uncertainty and limitations in planning/intelligence surfaces.
- Operator approval governance for non-trivial action paths.

## Near-term (credible)

- Better recommendation ranking informed by incident outcome history.
- Stronger evidence sufficiency scoring attached to each recommendation.
- Runbook linkage from similar-case retrieval to recommended actions.

## Longer-term (aspirational, not current claims)

- Operator-consented local model tuning for repetitive incident classes.
- Privacy-preserving recommendation memory exchange patterns.
- Mixed-channel advisory improvements (still non-canonical unless proven).

## Guardrails

- No deterministic RF/path certainty claims without direct evidence.
- No autonomous execution claim where actuators/policies are absent.
- No recommendation marketed as canonical truth.
