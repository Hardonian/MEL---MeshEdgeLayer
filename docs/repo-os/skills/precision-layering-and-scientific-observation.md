# Skill: Precision Layering and Scientific Observation

Use when adding/changing health calculators, telemetry interpretation, anomaly logic, recommendation logic, or incident summaries.

## Layered execution contract
- [ ] Semantics layer contracts are explicit.
- [ ] Telemetry layer metrics are named and sourced.
- [ ] Frequency/radio context is separated from direct causality claims.
- [ ] Physical/environment context is treated as enrichment unless proven causal.
- [ ] Sensor/spatial context usage is explicit and bounded.
- [ ] Algorithmic calculators expose evidence sufficiency and confidence posture.
- [ ] Mixed-channel truth posture is explicit (channel/path/support/privacy/assist flags).

## Scientific honesty checks
- [ ] Distinguish observed vs inferred vs estimated vs unknown.
- [ ] Avoid fake precision (do not imply unavailable measurements).
- [ ] Use deterministic heuristics/calculators before ML/LLM escalation.
- [ ] Mark proofpack/snapshot completeness as complete/partial/unavailable.
